# MVP — Minimum Viable Product

> O MVP (Minimum Viable Product) da Orizon Studio representa a primeira versão comercial da plataforma. Seu objetivo é validar o modelo de negócio, conquistar os primeiros clientes e criar uma base sólida para evolução contínua.

Versão: 1.0  
Status: Em elaboração

---

# Objetivo

O MVP deverá entregar valor real para escritórios de engenharia, arquitetura e design, resolvendo os principais problemas relacionados à captação de clientes, organização comercial e gestão dos projetos.

Nesta fase evitaremos funcionalidades complexas que aumentem o tempo de desenvolvimento sem gerar retorno imediato.

---

# Público-Alvo

O MVP será direcionado para:

- Engenheiros Civis
- Arquitetos
- Designers de Interiores
- Pequenos Escritórios
- Profissionais Autônomos

---

# Objetivos do MVP

- Validar o produto com clientes reais.
- Gerar os primeiros contratos.
- Reduzir o processo manual de atendimento.
- Organizar o relacionamento com clientes.
- Criar uma experiência premium desde o primeiro contato.

# Critérios para Lançamento

O MVP será considerado pronto quando atender aos seguintes requisitos:

- Landing Page publicada.
- Biblioteca de Projetos funcional.
- Configurador Inteligente operacional.
- Briefing Digital concluído.
- CRM funcional.
- Dashboard disponível.
- Portal do Cliente funcional.
- Gestão básica de documentos implementada.
- Ambiente multiempresa (multi-tenant).
- Autenticação e controle de acesso.
- Interface responsiva.
- Performance adequada.

# Considerações Finais

O MVP da Orizon Studio não busca entregar todas as funcionalidades imaginadas para a plataforma.

Seu objetivo é resolver os principais problemas dos profissionais de engenharia, arquitetura e design, permitindo validar rapidamente o produto com clientes reais.

Toda evolução futura será baseada no feedback dos usuários e na análise dos indicadores de uso, garantindo que a plataforma cresça de forma sustentável e orientada pelas necessidades do mercado.
